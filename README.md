# Raspberry PI Pico based digital synth

Fun personal project to combine software, hardware and 3D modelling/printing.

## Components

- Raspberry PI Pico
- Pico Audio module
- OLED screen
- 16-Channel analog multiplexer
- 11 potentiometers
- 3 rotary encoders
- 3 momentary push switches
- ON/OFF switch
- MIDI In
