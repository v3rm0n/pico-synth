# Raspberry PI Pico based digital synthesizer

Fun personal project to combine music, software, hardware and 3D modelling/printing.

## Components

- Raspberry PI Pico
- Pico Audio module
- 128x64 OLED screen
- 16-Channel analog multiplexer
- 11 potentiometers
- 3 rotary encoders
- 3 momentary push switches
- ON/OFF switch
- MIDI In
